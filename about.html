About the Project

The Drowsiness Detection System is a safety-focused application designed to prevent accidents caused by fatigue. By using real-time computer vision, the system monitors the user's eye movements and alerts them immediately if signs of sleepiness are detected.

Real-time Monitoring

Continuous face and eye tracking using high-speed YOLO algorithms.

Instant Alerts

Audio warnings to wake up the user when eyes are closed for too long.

Secure Access

SQL-backed user profiles and admin-only database management.

Technologies Used: Python | Flask | YOLO | OpenCV | SQLite3
